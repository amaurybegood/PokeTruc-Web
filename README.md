# PokéTruc — Exclusive Pokémon Card Illustrations

A fan-made website listing all Gen 1 Pokémon cards with **unique illustrations**, meaning cards only available in one language (Japanese, English, Chinese, or others).

🌐 **Live site**: https://amaurybegood.github.io/PokeTruc-Web/

---

## Features

- Browse all 151 Gen 1 Pokémon
- View cards with exclusive illustrations per language
- Search by name or number
- Dark mode
- Available in EN / FR / JA / KO / ZH

## Tech

Pure HTML / CSS / JS — no framework, no build step. Hosted on GitHub Pages.
